# Kubot
+ 이 봇은 관리 등 여러 기능이 있는 봇입니다.

## 소개
+ 관리와 재미를 목적으로 만들어졌습니다.
+ 당신의 디스코드방을 편리하게 만듭니다.

## 기능
+ 관리할 수 있는 명령어 and 정보 명령어와 -놀이 등 여려 명령어가 있습니다
+ -개발자로 개발자를 확인할 수 있습니다.

# 개발 관련
+ 개발 할때 쓴 파이썬버전, 라이브러리 버전을 적어두었습니다.

## 사용된 라이브러리
+ discord.py==1.7.2
+ datetime==4.3
+ Dtime==0.1

## 사용된 파이썬 버전
+ heroku=3.8.5
+ 본컴=3.8.5
